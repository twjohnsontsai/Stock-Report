5日 ╱ 20日集中度 — 完整說明與評估方式
① 集中度是什麼？計算方式
集中度（Concentration Index）由 WantGoo 計算，衡量「籌碼在過去 N 天內，是向主力集中還是向散戶分散」。
5日集中度：觀察最近 5 個交易日的主力買賣超相對成交量的加權偏態（正 = 主力淨買 → 籌碼集中；負 = 主力淨賣 → 籌碼分散）。
20日集中度：同上，但用 20 日窗口，反映中期趨勢，波動較小、方向較穩定。
兩條線的關係與位置比單一數值更重要。

③ 5日（快線）vs 20日（慢線）的使用方式
🟡 黃金交叉（5日由下穿越20日向上）
短線集中度開始超越中線，代表吸籌速度加快，主升段啟動的最重要信號。
配合買賣家數差由負轉正 = 更強確認。
🟣 死亡交叉（5日由上穿越20日向下）
短線分散速度超過中期，代表籌碼開始外流，出場警示信號。
若20日集中度同步下彎 = 主力確認退場，不宜守。
✅ 兩線均正且5日 > 20日
最強多頭格局：主力短中期均在集中籌碼，趨勢確立向上，持股最安全的區間。
⚠️ 兩線均負且5日 < 20日
最弱空頭格局：短中期籌碼均分散，主力持續離場，避免進場，持有者考慮減碼。
④ 8種集中度形態與操作建議
低位吸籌（深度築底）
5日在極深負值（<-7%），但 20日集中度已在改善（由負轉緩）。
意義：主力趁最後一波殺跌完成底部吸籌。操作：逢進場機會，等5日開始回升再分批買。
暴力恐嚇（趕人下車）
5日急跌速度遠快於20日（斜率差異大）。
意義：主力刻意製造恐慌，逼出短線持有者。操作：不追殺，有底部支撐時可留意反彈機會。
窒息鎖碼（沉澱期）
5日低位趨平（不再繼續跌），20日緩跌中。
意義：浮額洗盤接近尾聲，成交量極度萎縮，準備蓄勢。操作：耐心等待，直到5日開始回升。
初步換手（點火期）
5日開始向上穿越20日（兩線均仍在負值區）。
意義：主力開始點火，技術指標醞釀黃金交叉。操作：準備進場，分批布局，設好停損。
底部回升（準備上穿）
5日持續從低位上揚，逐漸靠近20日，即將形成黃金交叉。
意義：主升段啟動的前奏，是最佳進場準備窗口。操作：積極布局，等待黃金交叉確認後加碼。
低位回升（蓄勢期）
5日從深度負值持續回升，方向確認向上，但距20日仍有差距。
意義：正在蓄勢累積，需要時間讓20日也跟上。操作：小量試水，等5日持續向上且靠近20日再加碼。
強推換手（進攻期）
5日快速翻正並帶動20日上揚（兩線均正，5日 > 20日）。
意義：主力全面進攻，不惜代價拉升，即將進入噴發段。操作：續持或追買，停損設在20日集中度線下彎的位置。
高位洗盤（中繼站）
5日從正值高位回落，20日仍在上升中（兩線仍正，5日開始 < 20日）。
意義：拉高後短暫震盪洗籌，中期趨勢未變。操作：持有，不宜追高，等5日再次上揚確認後可加碼。
⑤ 結合其他指標提升準確性
集中度 + 買賣家數差
5日集中度轉正 + 買賣家數差由負轉正 = 雙重確認，散戶開始跟進，最強進場訊號。
集中度 + 大戶持股%
5日集中度暴跌（洗盤形態）+ 大戶持股%仍在上升 = 假跌真吸，是洗盤訊號而非真實出貨。
集中度 + 借券餘額
5日集中度回升 + 借券餘額同步減少 = 打壓結束 + 籌碼集中，底部最強組合訊號。

📌 浮籌日增減 = 法人淨賣超① + 融資淨賣超②；正（紅）=籌碼流向散戶，負（綠）=主力吸籌。
📌 散戶持股（校準） = 以 TDCC 週報（大戶持股%）為週頻錨點，換算散戶持股張數後，兩錨點間按日頻浮籌比例插值，得出每日散戶估算持股。
📌 主力持股（推算） = 總流通張數 − 散戶持股（校準），涵蓋大戶＋法人，為散戶的鏡像數據，兩者合計≈總流通。
📌 週誤差（驗證） = 每個 TDCC 錨點日才顯示：「日頻浮籌週累計」vs「TDCC實際週變化」的差距；誤差越小代表日頻計算越準確，可藉此驗證模型可靠度。滑鼠懸停可看詳情。
📌 週誤差偏大的常見原因：① 借券場外划拨（不過市場交易，不計入浮籌）② ETF 申贖（大量股票進出不走次級市場）③ 外資透過 ADR/GDR 的持股轉換 ④ 大宗交易（拍板成交，不計入一般成交量）。這些路徑都會造成散戶持股實際變化，但我們的日頻浮籌無法捕捉，屬於模型盲區。
